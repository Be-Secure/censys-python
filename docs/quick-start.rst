Quick Start 
===========

Assuming you have Python already, install the package:

.. tabs::

   .. tab:: from PyPi

      .. prompt:: bash

         pip install censys

   .. tab:: from Kali Linux

      .. prompt:: bash

         sudo apt install python3-censys

Configure your credentials:

.. tabs::

   .. tab:: Search API

      .. prompt:: bash

         censys config

   .. tab:: ASM API

      .. prompt:: bash
      
         censys asm config
